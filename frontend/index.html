How CrisisLens works

Every claim below is implemented in code you can read on GitHub. We're showing our work because trust in news aggregation has to be earned.

1. We ingest from multiple perspectives

Articles arrive every 15 minutes from Pan-Arab outlets (Al Jazeera, Al Arabiya), Western mainstream (BBC, AP, Washington Post, Deutsche Welle, France 24), and Israeli sources (Jerusalem Post). Each source carries a static trust weight (0.0–1.0) reflecting editorial track record.

2. We translate everything into a shared space

English headlines get machine-translated to Arabic; Arabic stays Arabic. Both versions are then encoded as 384-dimensional vectors with a multilingual embedding model so we can compare meaning across languages.

3. We pair articles likely to be about the same event

For each new article we find others with cosine similarity ≥ 0.70 from a different source published within 48 hours. Those pairs are candidates for a contradiction.

4. We classify pairs with a multilingual NLI model

Each candidate pair is sent through mDeBERTa-v3 (an open-source NLI model trained on 100+ languages) which returns probabilities for entailment / neutral / contradiction. Pairs above 55% contradiction confidence move forward.

5. We boost real disagreement, suppress framing differences

Pairs where headlines disagree on numbers (10 vs 7 casualties) get a confidence boost. Pairs that share most key terms — the "same story, different angle" pattern — get suppressed before they can become a flagged contradiction.

6. We score and surface the strongest signals

Final score = NLI contradiction confidence × highest-trust source. We never artificially weight perspectives — sources from any region can produce a real contradiction, and the strongest contradictions tend to show up across regions.

These categories describe regional/editorial provenance, not political stance — we make no claim that all sources within a category agree. The point of CrisisLens is to make disagreement visible, including disagreement within a category.

▾ What CrisisLens is not
CrisisLens is not a fact-checker — we don't say which source is right. We surface the disagreement and let you read both sides. Some "contradictions" are actually two outlets covering the same story differently, not factually disagreeing — we work to filter those out but they slip through. When in doubt, click through to both sources and judge for yourself.
